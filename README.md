# 🧑‍💻 OOP-2 Masterclass: Interactive C# Study Portal

A high-performance, bilingual (Bengali/English) study platform developed specifically for mastering Object-Oriented Programming (OOP) in C#. This portal provides interactive code examples, output tracing, and comprehensive documentation tailored for midterm success.

![Banner](https://img.shields.io/badge/Status-Production--Ready-success?style=for-the-badge&logo=react)
![Bilingual](https://img.shields.io/badge/Content-Bilingual--Support-blue?style=for-the-badge)

---

## ⚡ Key Features

- **🔍 Pro Search System**: Instantly find concepts across all 8 modules. Features match highlighting and automatic question expansion for lightning-fast access.
- **🇧🇩 Bilingual Core**: Seamlessly toggle between English and Bengali for all module basics and detailed question explanations.
- **📋 Copy-to-Clipboard**: One-click copying for all C# code snippets with visual confirmation.
- **🎨 Premium Aesthetics**: Features modern Glassmorphism UI, custom high-precision cursor, and smooth Framer Motion transitions.
- **📱 Ultra Responsive**: Optimized for desktops, tablets, and smartphones, ensuring study sessions are possible on any device.
- **🌑 Dark Mode Experience**: Deep slate and indigo color palette designed for low eye strain during long coding sessions.

## 🚀 Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v18 or higher)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/study-app.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```

### Development
Launch the local development server:
```bash
npm run dev
```

### Building for Production
Create an optimized production bundle:
```bash
npm run build
```
